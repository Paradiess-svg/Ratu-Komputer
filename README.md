# Proyek E-Commerce Ratu Komputer

Repositori ini berisi kode sumber untuk proyek website E-Commerce Ratu Komputer.

Website E-Commerce Ratu Komputer adalah sebuah platform ritel online yang dirancang untuk memfasilitasi penjualan produk-produk teknologi. Tujuan utamanya adalah untuk memindahkan proses bisnis konvensional klien (Ratu Komputer) ke platform digital yang terstruktur.

Sistem ini melayani dua peran utama:
1.  **Untuk Pelanggan:** Menyediakan antarmuka yang intuitif untuk menjelajahi produk, membebeli, dan simulasi rakit pc.
2.  **Untuk Admin:** Menyediakan dasbor terproteksi untuk mengelola data master seperti produk, cabang, karyawan/admin, hero, gallery produk, kategori, dan profil.

## ğŸ¯ Fitur Utama

### Sisi Pelanggan (Client-Side)
* **Katalog Produk:** Galeri produk dengan fitur pencarian dan kategori.
* **Proses Checkout:** Alur pemesanan via wa dan tak ada login/register.
* **Simulasi Rakit PC:** Sistem simulasi rakit pc untuk menampilkan simulasi merakit pc dengan ketentuan yang sudah ditetapkan.

### Sisi Admin (Admin-Side)
* **Manajemen Produk (CRUD):** Admin dan super admin dapat mengelola produk .
* **Manajemen Cabang:** Admin dan super admin dapat mengelola cabang.
* **Dasbor Admin:** Halaman utama yang dapat melihat daftar simulasi rakit pc dan admin/superadmin/owner di tabel karyawan.
* **Dasbor Super Admin:** Halaman utama untuk melihat daftar simulasi dan mengelola admin/superadmin (owner juga namun terproteksi rolenya tak bisa turun).
* **Manajemen Gallery Produk:** Admin dan super admin dapat mengelola foto produk.
* **Manajemen Hero:** Admin dan super admin dapat mengelola foto hero, promosi, dan produk terlaris termasuk linknya juga.
* **Manajemen Kategori:** Admin dan super admin dapat mengelola kategori produk.
* **Lihat profil:** Admin dan super admin dapat melihat profilnya sendiri.
